# Menu

Use Menus to help prioritize your neighbours’ attention on what’s important by deferring specialized
or secondary options.

## Basic

At its most basic, the `Menu` component is a pairing of a `button` element and a
`List` component. The menu is toggled open and closed when the button (the “menu
trigger”) is clicked:

## Trigger on hover

The component accepts a `triggerOnHover` prop that will open the menu when the
trigger is hovered or clicked.
