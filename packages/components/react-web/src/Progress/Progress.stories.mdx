# Progress

Use a progress indicator to give the user feedback about slow-running operations
such as submitting a form or loading data.

Use the **Determinate** variants for operations with a known duration or number
of steps. The **Indeterminate** variants should be used for operations expected
to last longer than 3 seconds but whose duration is unknown.
