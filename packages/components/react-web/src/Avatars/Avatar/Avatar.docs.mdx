# Avatar

Use the Avatar component to visually represent a user or organization. An
Avatar's size, shape, background color, and abbreviation may all be
customized. An [Image component](/docs/components-image) can be
provided as well.

Note that if an abbreviation is used, the component will select a text color
that provides sufficient contrast against its background color.

Related Avatars can be displayed together with the
[AvatarGroup component](/?path=/docs/layouts-avatars-avatargroup).
