# Space

Use the `space` element to provide consistency in the positioning of layouts or
components that use the Design System. The object provided by the `space`
element returns a scale of values calculated against a 16px base.

export const Space = styled.div`
  display: grid;
  grid-template-rows: 1fr auto;
  font-family: 'IBM Plex Mono';
  font-size: 0.625rem;
  align-items: center;
  justify-items: center;
  text-align: center;
  &::before {
    content: '';
    display: block;
    height: ${(props) => props.size};
    width: ${(props) => props.size};
    background: ${(props) => props.color};
  }
`;

## Usage

These object properties can be interpolated in CSS-in-JS or anywhere else that
specific pixel values are needed:

```js
return css`
  .my-component {
    padding: ${space.base} ${space.plus2}; /* Renders `padding: 8px 16px;` */
  }
`;
```
