## Typefaces & weights

The Compassion Design System uses **Montserrat** as its primary typeface. To
maintain a clean aesthetic, only the Regular, Extra Light, and Semibold weights
are included.

**Note:** The Design System aliases Montserrat as "Compassion Sans" in its
`@font-face` definitions to allow for easier upkeep.

## Modular scale

A constrained, purposeful set of font sizes is included in the Design System to
boost consistency within an interface and to assist in creating focused,
meaningful visual hierarchies. By default the modular scale uses a ratio of 1.33
between steps.
[_Read more about modular scales_](https://medium.com/sketch-app-sources/exploring-responsive-type-scales-cf1da541be54).

## Dynamic modular scales

A dynamic modular scale is provided out of the box to enable a tighter scale on
small screens and a wider scale on large screens. The dynamic scale acknowledges
the userâ€™s screen real estate while maintaining consistency and hierarchy.

Specific values on the dynamic scale can be accessed by passing a second
parameter to the `fontSize` function indicating the desired scale. If a second
parameter isn't provided the base scale is used:

```js
import { typography as t } from '@compassion-gds/elements';

t.fontSize(0, 'minus1'); // Returns base font size using the `minus1` ratio
t.fontSize(1); // Returns one step up the modular scale using the base ratio
t.fontSize(-1, 'plus1'); // Returns one step down the scale using the `plus1` ratio
```

## Body

The Design System recommends usage of a "system font stack" for body copy. Using
a system font stack allows users to see your content without the additional
network request required by a web font, which also eliminates a Flash of
Unstyled Text (FOUT). Additionally, the look and feel of content laid out with a
system font stack is highly legible and feels familiar to users because so much
app and OS content already uses these typefaces.

```js
return css`
  /*  Returns '-apple-system', BlinkMacSystemFont, 'avenir next', avenir,
   *          'segoe ui', 'helvetica neue', helvetica, Ubuntu, roboto, noto,
   *          arial, 'sans-serif' presently */
  font-family: ${typography.fonts.body};
`;
```
