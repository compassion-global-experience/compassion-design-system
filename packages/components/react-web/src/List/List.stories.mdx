# List

The List component should be used for continuous vertical groups of related
content. List items may contain text as well as rich content like images, icons,
or other markup. No matter the composition, each item should be appear
consistent with those around it and the items should be ordered logically.

Note that plaintext lists that donâ€™t require additional styling can be rendered
without this component using the semantically appropriate `ul` or `ol` element.

## Options

The padding and width of lists may be specified; alternating rows may be shown
with zebra stripes when a `stripeColor` is specified.
